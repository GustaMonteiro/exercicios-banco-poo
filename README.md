# Exercícios de POO - Conta Bancária e Banco

Este repositório contém exemplos de implementação de alguns exercícios da disciplina de Programação Orientada a Objetos (POO).

## Importante

- **Existem diversas formas de implementar estes exercícios.** As soluções apresentadas aqui são apenas uma das possíveis abordagens.
- Os números no início dos nomes dos diretórios fazem referência aos slides da disciplina.
- Apenas os exercícios relacionados ao tema de conta bancária e banco estão implementados neste repositório. Os demais exercícios não foram abordados.

## Conteúdo

Os exemplos abordam conceitos como:
- Criação de classes simples
- Uso de construtores
- Getters e setters
- Interfaces
- Uso de ArrayList para coleções
